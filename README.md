# Portfolio — Thomas Léon

Site personnel (React + Vite + Tailwind + Framer Motion). Déployé sur GitHub Pages.

## Développement
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Déploiement (GitHub Pages)
```bash
npm run deploy
```

> Le dépôt doit s'appeler **thomas-leon.github.io** pour être servi à la racine.
